이 폴더에 아래 mp3 파일을 넣으면 게임에서 자동으로 재생됩니다.
(파일이 없어도 게임은 정상 동작합니다 — 행사장 안정성을 위해 사운드는 선택 사항입니다.)

bubble.mp3   : 거품 생성 효과음
pop.mp3      : 계면이가 "뽁!" 하고 사라질 때
water.mp3    : 샤워기 물줄기 효과음
clear.mp3    : 미션 클리어
success.mp3  : 최종 클린 완료

※ 파일 이름은 위와 정확히 동일해야 합니다.
